# ناسازگاری نسخه‌های Python در requirements.txt و Dockerfile

## Raw Idea

فایل requirements.txt نیازمند Python 3.11+ است (به دلیل استفاده از async features و نوع‌دهی مدرن)، اما Dockerfile و render.yaml مشخصات نسخه Python را تعیین نمی‌کنند. این می‌تواند منجر به نصب ناقص وابستگی‌ها یا خطاهای runtime در محیط‌های استقرار شود. همچنین در فایل‌های مختلف (backend/Dockerfile, docker-compose.yml, render.yaml) هیچ اشاره‌ای به نسخه Python نشده است.
