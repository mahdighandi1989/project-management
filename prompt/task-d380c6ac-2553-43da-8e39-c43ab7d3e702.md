# [منطق] عدم وجود permission checking در auth pipeline

## Raw Idea

## 📋 شرح ناسازگاری
در pipeline `auth` یک ناسازگاری منطقی پیدا شد:

بر اساس توضیحات components، هیچ اشاره‌ای به مکانیزم permission checking در pipeline نشده است. database.py فقط وظیفه اتصال به دیتابیس را دارد و inspector_session.py مدل داده است. هیچ middleware یا dependency ای برای بررسی مجوزها قبل از mutation paths وجود ندارد.

## 💥 پیامد (impact)
