# [منطق] عدم coherence بین upload session و permission برای task_draft_id

## Raw Idea

## 📋 شرح ناسازگاری
در pipeline `auth` یک ناسازگاری منطقی پیدا شد:

سرویس upload_session از task_draft_id استفاده می‌کند اما هیچ بررسی نمی‌کند که کاربر جاری مجوز آپلود به آن task_draft_id خاص را دارد یا خیر. همچنین task_draft_id ممکن است متعلق به کاربر دیگری باشد.
