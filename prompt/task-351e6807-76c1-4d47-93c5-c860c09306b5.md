# [منطق] عدم تطابق بین prompt format و output parser در system_prompt.py

## Raw Idea

## 📋 شرح ناسازگاری
در pipeline `ai_llm` یک ناسازگاری منطقی پیدا شد:

system_prompt.py پرامپت‌های ترکیبی می‌سازد (build_combined_prompt) اما هیچ output parser مشخصی برای تفسیر خروجی مدل تعریف نشده است. همچنین مشخص نیست که format پرامپت (JSON، XML، Markdown) با کدام مدل‌های AI سازگار است.
