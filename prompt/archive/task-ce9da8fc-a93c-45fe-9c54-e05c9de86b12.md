## 📍 موقعیت دقیق در پروژه
_(file:line — symbol — snippet)_

- `backend/app/services/prompt_file_service.py` — `PromptFileService` — این یک سرویس جدید برای مدیریت فایل‌های پرامپت تسک‌ها است. بر اساس ساختار سطحی — توسط مجری تأیید شود.
  ```python
  import os
  from pathlib import Path
  
  class PromptFileService:
      PROMPT_DIR = Path("backend/storage/prompts")
  
      def __init__(self):
          self.PROMPT_DIR.mkdir(parents=True, exist_ok=True)
  
      def _get_file_path(self, task_id: str) -> Path:
          return self.PROMPT_DIR / f"task_{task_id}.md"
  
      def create_prompt_file(self, task_id: str, content: str):
          file_path = self._get_file_path(task_id)
          with open(file_path, "w", encoding="utf-8") as f:
              f.write(content)
  
      def update_prompt_file(self, task_id: str, content: str):
          file_path = self._get_file_path(task_id)
          if not file_path.exists():
              raise FileNotFoundError(f"Prompt file for task {task_id} not found.")
          with open(file_path, "w", encoding="utf-8") as f:
              f.write(content)
  
      def get_prompt_content(self, task_id: str) -> str:
          file_path = self._get_file_path(task_id)
          if not file_path.exists():
              raise FileNotFoundError(f"Prompt file for task {task_id} not found.")
          with open(file_path, "r", encoding="utf-8") as f:
              return f.read()
  ```

  import shutil
  
  class StorageService:
      def __init__(self, base_path: str = "backend/storage"):
          self.base_path = base_path
          self.prompts_path = os.path.join(base_path, "prompts")
          self.archive_path = os.path.join(self.prompts_path, "archive")
          os.makedirs(self.archive_path, exist_ok=True)
  
      async def move_file(self, source_relative_path: str, destination_relative_path: str):
          full_source = os.path.join(self.base_path, source_relative_path)
          full_dest = os.path.join(self.base_path, destination_relative_path)
          os.makedirs(os.path.dirname(full_dest), exist_ok=True)
          if not os.path.exists(full_source):
              raise FileNotFoundError(f"Source file not found: {full_source}")
          shutil.move(full_source, full_dest)
          return full_dest

## ⚠️ ریسک‌ها و موارد احتیاط
1.  **Database Schema Migration:** افزودن فیلد `priority` به مدل `SystemPrompt` در `backend/app/models/system_prompt.py` نیاز به یک migration دیتابیس دارد. عدم اجرای صحیح migration می‌تواند منجر به خطاهای دیتابیس در محیط‌های موجود شود و نیاز به هماهنگی با تیم DevOps دارد.
2.  **Backward Compatibility:** اگر `Cloud Code` یا سایر ابزارهای داخلی قبلاً به API `system_prompts` بدون در نظر گرفتن اولویت متکی بوده‌اند، تغییر در ساختار پاسخ یا رفتار مرتب‌سازی ممکن است باعث شکست در آن‌ها شود. باید اطمینان حاصل شود که endpointهای موجود تحت تأثیر قرار نمی‌گیرند یا تغییرات به صورت سازگار با عقب (backward compatible) اعمال می‌شوند.
3.  **Performance Impact:** مرتب‌سازی پرامپت‌ها بر اساس اولویت در دیتابیس (به خصوص اگر تعداد پرامپت‌ها زیاد باشد) می‌تواند بر عملکرد کوئری‌ها تأثیر بگذارد. نیاز به بررسی ایندکس‌گذاری روی فیلد `priority` در مدل `SystemPrompt` وجود دارد تا از افت عملکرد جلوگیری شود.
4.  **Impact on `backend/app/services/ai_manager.py` and `backend/app/services/oversight_service.py`:** این سرویس‌ها ممکن است دارای منطق پیچیده‌ای برای پردازش پرامپت‌ها باشند. تغییر در نحوه بازیابی و ترتیب پرامپت‌ها می‌تواند عوارض جانبی ناخواسته‌ای در جریان‌های کاری موجود ایجاد کند و نیاز به تست‌های جامع دارد.
