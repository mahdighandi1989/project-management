# duplicate prompt builder در runtime_executor و oversight_strong_prompt

## Raw Idea

دو فایل `backend/app/services/runtime_executor.py` و `backend/app/services/oversight_strong_prompt.py` هر دو تابع `build_prompt` با منطق متفاوت دارند. اولی برای اجرای runtime پروژه‌ها و دومی برای ساخت پرامپت‌های قوی در oversight استفاده می‌شود. این duplication باعث confusion در نگهداری و به‌روزرسانی می‌شود و ممکن است باگ‌های ظریف در رفتار پرامپت‌ها ایجاد کند.
