## 💡 نمونه‌های قبل/بعد
**قبل: عدم مدیریت خطا**

_قبل:_
```
@router.get('/projects')
async def get_projects():
    return await service.get_all()
```

_بعد:_
```
@router.get('/projects')
async def get_projects():
    try:
        return await service.get_all()
    except Exception as e:
        logger.error('Failed to get projects', exc_info=e)
        raise HTTPException(status_code=500, detail='Internal server error')
```
