## Raw Idea

```

**Status:** `pending` (0%)
**Scope:** در این مرحله، متن استخراج‌شده بر اساس انتخاب اولیه کاربر (ثبت ایده برای پروژه موجود یا ثبت پروژه جدید) مسیریابی می‌شود. اگر کاربر پروژه جدید خواسته باشد، متن به موتور خالق (موجود در سیستم فعلی) ارسال می‌شود. اگر برای پروژه موجود باشد، متن به بخش تسک‌های آن پروژه اضافه می‌شود. این مرحله شامل منطق تصمیم‌گیری و ارسال به ماژول‌های مربوطه است.
**Excerpt:**
```
بعد از انتخاب اینکه درخواستم برای ثبت ایده برای یک پروژه موجود یا ثبت یک پروژه جدید و ... است حسب آن، بتوانم با ضبط صدا و گفتن ایده ... اگر مربوط به ایده ساخت پروژه جدید بود به موتور خالق میره متن استخراج شده، و اونجاروال عادی خودشو طی میکنه
