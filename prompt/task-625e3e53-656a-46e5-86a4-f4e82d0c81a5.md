# Duplicate project creation endpoints: simple_projects vs unified_api vs orchestrator

## Raw Idea

سه endpoint مجزا برای ایجاد پروژه وجود دارد: POST /api/simple/projects/create (simple_projects.py), POST /api/unified/projects/create (unified_api.py), و POST /orchestrator/smart-setup (orchestrator.py). هر سه مسیر منطق مشابهی برای ساخت پروژه با AI دارند اما با signature و response shape متفاوت. این duplication باعث confusion در frontend می‌شود (creator page از simple_projects استفاده می‌کند، diagrams page از unified_api، و oversight page از orchestrator). همچنین maintenance burden ایجاد می‌کند چون تغییر در یک endpoint باید در سه جا replicate شود.
