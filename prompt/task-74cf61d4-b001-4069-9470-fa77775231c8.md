# mismatch بین reactflow v11 و react 18 در package.json

## Raw Idea

در frontend/package.json، reactflow نسخه '^11.11.4' و react نسخه '^18.2.0' تعریف شده‌اند. reactflow v11 به react 17 وابسته است و با react 18 ناسازگاری دارد — این باعث خطاهای hydration و رندر در کامپوننت‌های ReactFlow (مثل صفحهٔ project detail) می‌شود. شواهد: فایل frontend/src/app/projects/[id]/page.tsx در خطوط 9-19 از reactflow import می‌کند و از useNodesState, useEdgesState استفاده می‌کند که در v11 با react 18 hook rules را نقض می‌کنند.
